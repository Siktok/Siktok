# Sobre mi

Desarrollador **Full Stack .NET** con más de **6 años de experiencia** en desarrollo de aplicaciones web, APIs REST, microservicios y soluciones enterprise.

Estoy especializado en **C#**, **.NET / ASP.NET Core**, **Angular**, **SQL Server**, **Entity Framework Core**, arquitectura por capas, principios **SOLID**, **DDD** y **Clean Architecture**.

Actualmente estoy orientando mi perfil hacia el desarrollo de software asistido por IA: **Spec-Driven Development**, uso avanzado de **LLMs**, **RAG**, **GitHub Copilot**, automatización, orquestación de agentes IA y documentación técnica como parte del ciclo de vida del software.
